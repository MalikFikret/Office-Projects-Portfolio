OOP ARCHITECTURE - ABSTRACTION

The AIPlayer class abstracts away the complex math of Depth-First Search (DFS) and recursive backtracking algorithms.

Hiding Complexity

The main game loop interacts with the AI through simple public methods (like reset or getPosition), completely unaware of the complex Stack and HashSet logic running inside.

Simple Interaction

We treat the AI logic as a "black box" we input the maze, and it outputs the next move. The internal pathfinding calculation is hidden from the rest of the system.
